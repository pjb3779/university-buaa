# university-buaa

소프트웨어학 전공 과정에서 수행한 주요 전공 과목의 실습, 과제, 프로젝트를 정리한 저장소입니다.

자료구조, 알고리즘, 운영체제, 컴파일러, 네트워크, 데이터베이스, 소프트웨어공학 등 전공 핵심 과목을 학습하며 작성한 코드와 과제 결과물을 학기별로 정리했습니다.  
단순 이론 정리보다는 실제 구현, 문제 풀이, 시스템 동작 원리 이해, 실습 기반 학습 경험을 기록하는 데 목적이 있습니다.
