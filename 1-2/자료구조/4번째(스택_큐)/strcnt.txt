1. strcat (String Concatenate)
strcat 함수는 두 문자열을 연결하여 하나의 문자열로 만듭니다. 이 함수는 두 번째 문자열을 첫 번째 문자열의 끝에 덧붙입니다.

사용법

char *strcat(char *dest, const char *src);
dest: 결과 문자열이 저장될 목적지 버퍼입니다. dest는 충분한 공간을 확보하고 있어야 하며, src를 덧붙인 결과가 dest에 저장됩니다.

src: 덧붙여질 소스 문자열입니다.

strcat의 동작은 dest 문자열의 끝에 src 문자열을 덧붙이는 방식입니다. 이때 dest는 널 종료 문자(\0)로 끝나야 하며, src는 그대로 덧붙여집니다.

예시

char str1[50] = "Hello, ";
char str2[] = "World!";
strcat(str1, str2);
printf("%s\n", str1);  // 출력: Hello, World!
특징
동작: dest의 끝을 찾아서 src 문자열을 덧붙입니다.

주의: dest의 크기는 src의 크기와 합쳐져도 버퍼를 초과하지 않도록 충분히 커야 합니다. 그렇지 않으면 버퍼 오버플로우가 발생할 수 있습니다.

2. strncat (String Concatenate with Limit)
strncat 함수는 strcat과 거의 동일하지만, 덧붙일 문자열의 최대 길이를 제한할 수 있습니다. 이 함수는 두 번째 문자열에서 최대 n개의 문자를 덧붙입니다. 이를 통해 dest 버퍼가 초과되지 않도록 보호할 수 있습니다.

사용법

char *strncat(char *dest, const char *src, size_t n);
dest: 결과 문자열이 저장될 목적지 버퍼입니다.

src: 덧붙여질 소스 문자열입니다.

n: src에서 덧붙일 최대 문자 수입니다. src의 길이가 n보다 길 경우 n만큼만 덧붙입니다.

예시
char str1[50] = "Hello, ";
char str2[] = "World!";
strncat(str1, str2, 3);  // str2에서 최대 3개의 문자만 덧붙입니다.
printf("%s\n", str1);  // 출력: Hello, Wor
특징
동작: strncat은 src 문자열에서 최대 n개의 문자를 dest의 끝에 덧붙입니다.

주의: n은 src 문자열의 크기보다 작거나 같아야 하며, 덧붙인 후 dest 문자열의 끝에 널 종료 문자(\0)를 자동으로 추가합니다.

버퍼 보호: strncat은 n을 지정하여 덧붙일 문자의 수를 제한할 수 있기 때문에, dest 버퍼가 초과되지 않도록 보호할 수 있습니다.
